# 配置swap内存

检查系统是否已经配置过swap，如果该命令没有返回出结果，则代表该系统尚未配置过swap
```
swapon -s
```

创建swap文件
```
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
```

启用swap文件
```
# 临时
sudo swapon /swapfile

# 永久
sudo echo "/swapfile   swap    swap    sw  0   0" >> /etc/fstab
```

# 安装docker

# we-mp-rss

项目地址：[we-mp-rss](https://github.com/rachelos/we-mp-rss)

配置方法：
```
sudo mkdir /dockerdata/we-mp-rss
sudo docker run -d --name we-mp-rss -p 8001:8001 -v /dockerdata/we-mp-rss:/app/data ghcr.io/rachelos/we-mp-rss:1.4.5    # 直接使用latest代替1.4.5无法访问

sudo docker update --memory 1g we-mp-rss
sudo docker update --memory-swap 4g we-mp-rss

-m 1g --shm-size=4g
--memory-swap 4g
--memory-swappiness 100
```
